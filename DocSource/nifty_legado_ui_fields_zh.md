# Nifty 书源字段（JSON 键 -> 书源管理 WebUI 中文名）

## 基础
- `bookSourceType` -> **源类型**
- `bookSourceUrl` -> **源域名**
- `bookSourceName` -> **源名称**
- `bookSourceGroup` -> **源分组**
- `bookSourceComment` -> **源注释**
- `bookUrlPattern` -> **链接验证**
- `header` -> **请求头**

## 搜索
- `searchUrl` -> **搜索地址**
- `ruleSearch.bookList` -> **列表规则**
- `ruleSearch.name` -> **书名规则**
- `ruleSearch.kind` -> **分类规则**
- `ruleSearch.lastChapter` -> **最新章节**
- `ruleSearch.intro` -> **简介规则**
- `ruleSearch.bookUrl` -> **详情地址**

## 发现
- `exploreUrl` -> **发现地址**
- `ruleExplore.bookList` -> **列表规则**
- `ruleExplore.name` -> **书名规则**
- `ruleExplore.kind` -> **分类规则**
- `ruleExplore.lastChapter` -> **最新章节**
- `ruleExplore.intro` -> **简介规则**
- `ruleExplore.bookUrl` -> **详情地址**

## 详情
- `ruleBookInfo.name` -> **书名规则**
- `ruleBookInfo.kind` -> **分类规则**
- `ruleBookInfo.lastChapter` -> **最新章节**
- `ruleBookInfo.intro` -> **简介规则**
- `ruleBookInfo.tocUrl` -> **目录地址**

## 目录
- `ruleToc.chapterList` -> **列表规则**
- `ruleToc.chapterName` -> **章节名称**
- `ruleToc.chapterUrl` -> **章节地址**

## 正文
- `ruleContent.content` -> **正文规则**

## 其他
- `enabled` -> **启用搜索**
- `enabledExplore` -> **启用发现**
- `weight` -> **搜索权重**
